\section{Aufgabe 14: Listen
\index{Doppelt-verkettete Liste}
\footcite[entnommen aus Algorithmen und
Datenstrukturen, Präsenzübung 4, Universität Bayreuth, Aufgabe 14]{aud:pu:7}}

Betrachten Sie folgendes Klassendiagramm\index{Klassendiagramm}, das
doppelt-verkettete Listen spezifiziert. Die Assoziation
\liJavaCode{head} zeigt auf das erste Element der Liste. Die
Assoziationen \liJavaCode{previous} und \liJavaCode{next} zeigen auf das
vorherige bzw. folgende Element.
\footcite[Thema 1 Aufgabe 1]{examen:66112:2005:03}

\begin{center}

\noindent
Implementieren Sie die doppelt-verketteten Listen in einer geeigneten
objektorientierten Sprache (\zB Java oder C++), das heißt:

\begin{enumerate}

%%
% (a)
%%

\item Implementieren Sie die Klasse \liJavaCode{ListElem}. Die Methode
\liJavaCode{insert} ordnet eine ganze Zahl \liJavaCode{i} in eine
aufsteigend geordnete doppelt-verkettete Liste \liJavaCode{l} an die
korrekte Stelle ein. Sei \zB das Objekt \liJavaCode{l} eine
Repräsentation der Liste \liJavaCode{[0, 2, 2, 6, 8]} dann liefert
\liJavaCode{l.insert(3)} eine Repräsentation der Liste \liJavaCode{[0,
2, 2, 3, 6, 8]}.

\begin{liAntwort}
\liJavaExamen{66112}{2005}{03}{ListElem}
\end{liAntwort}

%%
% (b)
%%

\item Implementieren Sie die Klasse \liJavaCode{DoublyLinkedList}, wobei
die Methode \liJavaCode{insert} eine Zahl \liJavaCode{i} in eine
aufsteigend geordnete Liste einordnet. Die Methode \liJavaCode{check}
überprüft, ob eine Liste korrekt verkettet ist, \dh ob für jedes
ListElem-Objekt \liJavaCode{o}, das über den \liJavaCode{head} der Liste
erreichbar ist, der Vorgänger des Nachfolgers von \liJavaCode{o} gleich
\liJavaCode{o} ist.
\end{enumerate}
