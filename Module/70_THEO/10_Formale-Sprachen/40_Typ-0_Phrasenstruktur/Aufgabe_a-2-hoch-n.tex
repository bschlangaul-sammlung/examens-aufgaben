\section{Ableitung einer Phrasenstrukturgrammatik
\index{Unbeschränkte Sprache}
\footcite[Seite 15-16]{thoe:fs:3}}

Die folgende Grammatik erzeugt die Sprache \bAusdruck{a^{2^n}}{n \in
\mathbb{N}}, nicht kontextsensitiv ist:

\bGrammatik{variablen={S, L, R, D}, alphabet={a}}
\begin{bProduktionsRegeln}
S -> L a R,
L -> L D | EPSILON,
D a -> a a D,
D R -> R,
R -> EPSILON,
\end{bProduktionsRegeln}

\noindent
Leiten Sie das Wort $a^8$ ab.

\begin{bAntwort}
\bAbleitung{LaR
-> LDaR
-> LDDaR
-> LDDDaR
-> DDDaR
-> DDaaDR
-> DaaDaDR
-> aaDaDaDR
-> aaaaDDaDR
-> aaaaaaaaDDDR
-> aaaaaaaaDDR
-> aaaaaaaaR
-> aaaaaaa}
\end{bAntwort}
