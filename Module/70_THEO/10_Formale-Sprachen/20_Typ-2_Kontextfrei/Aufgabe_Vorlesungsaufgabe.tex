\begin{enumerate}
\item Erstelle eine (deterministische) Grammatik für Palindrome, für die
ein DPDA existiert.\index{Kontextfreie Sprache}\footcite[Seite 34]{theo:fs:2}

\bAusdruck[L]{w \$ w^R}{w \in (a|b)^*}

\item Wandle diese Grammatik in einen DPDA um.
\end{enumerate}

Überführe die folgenden kontextfreien Grammatiken in CNF\footcite[Seite
37]{theo:fs:2}

\begin{bProduktionsRegeln}
S -> ABC,
A -> aCD,
B -> bCD,
C -> D | epsilon,
D -> C
\end{bProduktionsRegeln}
